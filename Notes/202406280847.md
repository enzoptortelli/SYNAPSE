# Definition 
Dois eventos $A, B$ serão disjuntos se $A \cap B = \emptyset$. Além disso, os eventos $A_1, A_2, \dots$ são ditos disjuntos dois a dois, ou mutualmente exclusivos, se $A_i \cap A_j = \emptyset \quad \forall i \neq j$.
