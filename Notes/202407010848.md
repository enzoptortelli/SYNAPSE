# Definition 
Uma cadeia de Markov (CM) é um processo estocástico que satisfaz a propriedade markoviana; isto é, $$P(X_{n+1} = j | X_n = x_n, X_{n-1} = x_{n-1}, \dots, X_0 = x_0) = P(X_{n+1} = j | X_n = x_n)$$
